| 概念     | 把已经编写好的类打包放在服务器上执行。（区别于在客户端上运行。） |
| -------- | ------------------------------------------------------------ |
| 技术基础 | 远程方法调用（RMI）（技术基础：对象的序列化；远程过程调用（RPC）） |

组成：

1. SessionBean
2. Entity Bean
3. MessageDriven Bean

EJB似乎是一个老技术，更多的人开始用spring。看了一堆别人的博客还有百度百科，也没实际搞懂它。或许要有实际开发经验，才能理解这个技术更替的意义吧。
