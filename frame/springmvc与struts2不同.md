# 两个框架的三大区别

| 区别     | springmvc                                                    | struts2                                                 |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------- |
| 入口     | 前端控制器（servlet）                                        | 核心过滤器（filter）                                    |
| 开发方式 | 基于方法<br/>单例为主                                        | 基于类<br>只能多例                                      |
| 参数传递 | 请求参数传到方法形参;<br/>数据和视图封装成ModelAndView对象<br>视图解析器：JSTL表达式 | 采用值栈存储请求和响应的数据<br/>视图解析器：OGNL表达式 |
